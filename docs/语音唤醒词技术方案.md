# 语音唤醒词 + 本地语音识别技术方案

## 1. 背景与目标

SpeakAssist 悬浮窗当前采用点击触发语音输入，用户需要手动点击悬浮按钮才能发起语音命令。本方案增加「语音唤醒词」功能：用户说出唤醒词（如"小噜小噜"）后，悬浮窗自动进入收音状态。

同时，将现有**百度 REST API 语音识别**替换为**本地离线语音识别（Vosk）**，彻底摆脱网络依赖。

### 核心约束

- **离线优先**：唤醒词检测 + 命令识别全部在设备本地完成
- **零成本**：不使用任何付费服务，不消耗百度 API 额度
- **低功耗**：悬浮窗常驻监听，功耗需可控
- **可集成**：与现有 SpeakAssist 架构无缝集成，不破坏现有功能

---

## 2. 技术方案确定：Vosk

### 2.1 为什么选 Vosk

| 对比项 | Vosk（最终选择） | 百度 REST API（原有） |
|--------|-----------------|---------------------|
| **费用** | 完全免费 | 每天500次额度 |
| **网络** | 完全离线 | 必须联网 |
| **延迟** | <100ms | 依赖网络，500ms~2s |
| **唤醒词检测** | 可做 | 不适合（API调用制不适合持续监听） |
| **模型大小** | ~50MB（中文） | 无（云端） |
| **精度** | 中等（对短命令够用） | 高 |
| **稳定性** | 取决于手机性能 | 取决于网络质量 |

### 2.2 Vosk 是什么

Vosk 是 Carnegie Mellon University（卡内基梅隆大学）开源的离线语音识别引擎：

- **官网**：https://alphacephei.com/vosk
- **开源协议**：Apache 2.0，完全免费商用
- **模型大小**：中文模型约 50MB（已含词库）
- **支持平台**：Android、iOS、Linux、Windows、Raspberry Pi
- **语言支持**：中文（含普通话和多种方言）、英文等 20+ 种语言
- **输出格式**：逐词时间戳 + 置信度，支持流式识别

### 2.3 为什么不用 Porcupine 做唤醒词

Porcupine 是专门做唤醒词的，精度高功耗低，但**中文模型需要付费商业授权**。毕业设计没钱，所以：

- **唤醒词检测**：用 Vosk（通用 ASR 可以检测任意词，包括唤醒词）
- **命令识别**：同样用 Vosk

两者统一用一个引擎，减少复杂度。

### 5.2 步骤2：创建 VoskRecognizerManager

```kotlin
package com.example.speech

import android.content.Context
import android.util.Log
import org.vosk.Model
import org.vosk.Recognizer
import java.io.File
import java.io.IOException

/**
 * Vosk 语音识别管理器
 * 负责加载模型、创建 Recognizer、处理识别结果
 */
class VoskRecognizerManager(private val context: Context) {

    companion object {
        private const val TAG = "VoskRecognizerManager"
        private const val MODEL_NAME = "vosk-model-cn-0.1"
        private const val SAMPLE_RATE = 16000f
    }

    private var model: Model? = null
    private var recognizer: Recognizer? = null

    /**
     * 加载模型（从 assets 目录）
     * 注意：首次调用需要复制文件，可能需要1-2秒
     */
    fun loadModel(): Boolean {
        try {
            val modelPath = getModelPath()
            if (!File(modelPath).exists()) {
                Log.w(TAG, "模型不存在: $modelPath，尝试从 assets 复制")
                copyModelFromAssets()
            }
            model = Model(modelPath)
            Log.i(TAG, "Vosk 模型加载成功: $modelPath")
            return true
        } catch (e: IOException) {
            Log.e(TAG, "模型加载失败", e)
            return false
        } catch (e: Exception) {
            Log.e(TAG, "模型加载异常", e)
            return false
        }
    }

    /**
     * 从 assets 复制模型到 app 私有目录
     */
    private fun copyModelFromAssets() {
        val assetManager = context.assets
        val destDir = File(context.filesDir, MODEL_NAME)

        // 逐个复制目录下的文件
        copyDirectory(assetManager, MODEL_NAME, destDir)
        Log.i(TAG, "模型已从 assets 复制到: ${destDir.absolutePath}")
    }

    private fun copyDirectory(assetManager: android.content.res.AssetManager, srcPath: String, destDir: File) {
        destDir.mkdirs()
        try {
            val files = assetManager.list(srcPath) ?: return
            for (file in files) {
                val src = "$srcPath/$file"
                val destFile = File(destDir, file)
                if (assetManager.list(src).isNullOrEmpty()) {
                    // 是文件，直接复制
                    assetManager.open(src).use { input ->
                        destFile.outputStream().use { output ->
                            input.copyTo(output)
                        }
                    }
                } else {
                    // 是目录，递归复制
                    copyDirectory(assetManager, src, destFile)
                }
            }
        } catch (e: Exception) {
            Log.e(TAG, "复制模型文件失败: $srcPath", e)
        }
    }

    private fun getModelPath(): String {
        return "${context.filesDir.absolutePath}/$MODEL_NAME"
    }

    /**
     * 创建 Recognizer（每次识别任务创建一个）
     * @param maxResultCount 最多返回几个候选结果
     */
    fun createRecognizer(maxResultCount: Int = 1): Recognizer? {
        val m = model ?: return null
        return try {
            Recognizer(m, SAMPLE_RATE, maxResultCount).apply {
                recognizer = this
            }
        } catch (e: Exception) {
            Log.e(TAG, "创建 Recognizer 失败", e)
            null
        }
    }

    /**
     * 处理音频数据（16bit PCM，16kHz 单声道）
     * @return 识别结果字符串，如果还在识别中返回 null
     */
    fun acceptAudio(audioData: ByteArray): String? {
        val rec = recognizer ?: return null
        return try {
            val isFinal = rec.acceptWaveForm(audioData, audioData.size)
            if (isFinal) {
                rec.result.text
            } else {
                // 部分结果
                rec.partialResult.partial
            }
        } catch (e: Exception) {
            Log.e(TAG, "识别失败", e)
            null
        }
    }

    /**
     * 获取最终识别结果
     */
    fun finalResult(): String? {
        return recognizer?.result?.text
    }

    /**
     * 重置识别器状态
     */
    fun reset() {
        recognizer?.reset()
    }

    /**
     * 释放资源
     */
    fun destroy() {
        recognizer?.close()
        recognizer = null
        model?.close()
        model = null
        Log.d(TAG, "VoskRecognizerManager 已销毁")
    }
}
```

### 5.3 步骤3：创建 WakeWordListeningManager

这是核心模块，负责：
1. 持续从麦克风采集音频
2. 送入 Vosk 识别
3. 检测是否说了唤醒词
4. 检测到后等待命令识别

```kotlin
package com.example.speech

import android.Manifest
import android.content.Context
import android.content.pm.PackageManager
import android.media.AudioFormat
import android.media.AudioRecord
import android.media.MediaRecorder
import android.os.Handler
import android.os.Looper
import android.util.Log
import androidx.core.app.ActivityCompat
import kotlinx.coroutines.*
import kotlinx.coroutines.flow.MutableStateFlow
import kotlinx.coroutines.flow.StateFlow

/**
 * 唤醒词监听管理器
 * 持续监听麦克风，检测到唤醒词后触发后续流程
 */
class WakeWordListeningManager(private val context: Context) {

    companion object {
        private const val TAG = "WakeWordListeningManager"

        // 音频参数（必须与 Vosk 模型采样率一致）
        private const val SAMPLE_RATE = 16000
        private const val CHANNEL_CONFIG = AudioFormat.CHANNEL_IN_MONO
        private const val AUDIO_FORMAT = AudioFormat.ENCODING_PCM_16BIT

        // 唤醒词
        private const val WAKE_WORD = "小噜小噜"

        // 静音检测参数
        private const val SILENCE_THRESHOLD = 500        // 静音阈值（RMS）
        private const val SILENCE_DURATION_MS = 1500L    // 静音持续多久认为说完
        private const val MAX_LISTENING_MS = 10000L      // 最大收音时长
    }

    // 监听状态
    enum class State {
        IDLE,           // 待机中，等待唤醒
        WAKE_DETECTED,  // 唤醒词检测到
        LISTENING_CMD,  // 等待命令
        RECOGNIZING,    // 识别中
        COMPLETED,      // 完成
        ERROR           // 错误
    }

    private val _state = MutableStateFlow(State.IDLE)
    val state: StateFlow<State> = _state

    private var voskManager: VoskRecognizerManager? = null
    private var audioRecord: AudioRecord? = null
    private var listeningJob: Job? = null
    private val scope = CoroutineScope(Dispatchers.IO + SupervisorJob())
    private val handler = Handler(Looper.getMainLooper())

    // 回调
    var listener: Listener? = null

    interface Listener {
        fun onWakeWordDetected()           // 唤醒词检测到
        fun onCommandRecognized(text: String) // 命令识别完成
        fun onError(message: String)        // 出错
        fun onStateChanged(state: State)   // 状态变化
    }

    /**
     * 初始化，加载模型
     */
    fun init(): Boolean {
        voskManager = VoskRecognizerManager(context)
        return voskManager?.loadModel() == true
    }

    /**
     * 开始监听（持续监听唤醒词）
     */
    fun startListening() {
        if (_state.value != State.IDLE) {
            Log.w(TAG, "当前状态不是 IDLE，无法开始监听: ${_state.value}")
            return
        }

        if (!checkPermission()) {
            listener?.onError("缺少录音权限")
            return
        }

        listeningJob = scope.launch {
            startAudioLoop()
        }
    }

    /**
     * 停止监听
     */
    fun stopListening() {
        listeningJob?.cancel()
        listeningJob = null
        stopRecording()
        _state.value = State.IDLE
        listener?.onStateChanged(State.IDLE)
    }

    /**
     * 释放资源
     */
    fun destroy() {
        stopListening()
        voskManager?.destroy()
        voskManager = null
        scope.cancel()
    }

    // ==================== 私有方法 ====================

    private fun checkPermission(): Boolean {
        return ActivityCompat.checkSelfPermission(context, Manifest.permission.RECORD_AUDIO) \
                == PackageManager.PERMISSION_GRANTED
    }

    private suspend fun startAudioLoop() {
        // 创建 Recognizer 用于唤醒词检测
        val recognizer = voskManager?.createRecognizer(1) ?: run {
            listener?.onError("无法创建语音识别器")
            return
        }

        // 初始化 AudioRecord
        val bufferSize = AudioRecord.getMinBufferSize(SAMPLE_RATE, CHANNEL_CONFIG, AUDIO_FORMAT)
        if (bufferSize == AudioRecord.ERROR || bufferSize == AudioRecord.ERROR_BAD_VALUE) {
            listener?.onError("无法获取录音缓冲区大小")
            return
        }

        audioRecord = try {
            AudioRecord(
                MediaRecorder.AudioSource.MIC,
                SAMPLE_RATE,
                CHANNEL_CONFIG,
                AUDIO_FORMAT,
                bufferSize * 2
            )
        } catch (e: SecurityException) {
            listener?.onError("缺少录音权限")
            return
        }

        if (audioRecord?.state != AudioRecord.STATE_INITIALIZED) {
            listener?.onError("录音初始化失败")
            return
        }

        audioRecord?.startRecording()
        Log.d(TAG, "开始监听...")

        val buffer = ByteArray(bufferSize)
        var silenceStartTime = 0L
        var hasSpeech = false
        var listeningStartTime = System.currentTimeMillis()

        // 唤醒词检测循环
        while (isActive && audioRecord?.recordingState == AudioRecord.RECORDSTATE_RECORDING) {
            val read = audioRecord?.read(buffer, 0, buffer.size) ?: -1
            if (read <= 0) continue

            // 送入 Vosk 识别
            val result = recognizer.acceptWaveForm(buffer, read)

            // 计算音量
            val rms = calculateRms(buffer, read)

            when (_state.value) {
                State.IDLE -> {
                    // 检测唤醒词
                    if (result != null && result.isNotBlank()) {
                        Log.d(TAG, "Vosk 识别结果: $result")
                        if (result.contains(WAKE_WORD)) {
                            Log.i(TAG, "唤醒词检测到！")
                            hasSpeech = false
                            silenceStartTime = 0
                            updateState(State.WAKE_DETECTED)
                            handler.post { listener?.onWakeWordDetected() }
                        }
                    }

                    // 超时停止（防止一直空转耗电）
                    if (System.currentTimeMillis() - listeningStartTime > MAX_LISTENING_MS) {
                        Log.d(TAG, "监听超时，重置")
                        recognizer.reset()
                        listeningStartTime = System.currentTimeMillis()
                    }
                }

                State.WAKE_DETECTED, State.LISTENING_CMD -> {
                    // 等待命令，静音检测
                    if (rms > SILENCE_THRESHOLD) {
                        hasSpeech = true
                        silenceStartTime = 0
                    } else if (hasSpeech) {
                        if (silenceStartTime == 0L) {
                            silenceStartTime = System.currentTimeMillis()
                        }
                        val silenceDuration = System.currentTimeMillis() - silenceStartTime
                        if (silenceDuration >= SILENCE_DURATION_MS) {
                            // 静音足够久，认为说完了
                            val finalResult = recognizer.finalResult() ?: ""
                            Log.d(TAG, "命令识别完成: $finalResult")
                            if (finalResult.isNotBlank()) {
                                updateState(State.RECOGNIZING)
                                handler.post {
                                    listener?.onCommandRecognized(finalResult)
                                }
                            } else {
                                // 没识别到内容，回到 IDLE
                                Log.d(TAG, "未识别到命令内容")
                                updateState(State.IDLE)
                                recognizer.reset()
                                hasSpeech = false
                                silenceStartTime = 0
                                listeningStartTime = System.currentTimeMillis()
                            }
                        }
                    }
                }

                else -> {}
            }
        }

        stopRecording()
    }

    private fun stopRecording() {
        try {
            audioRecord?.stop()
            audioRecord?.release()
        } catch (e: Exception) {
            Log.w(TAG, "停止录音失败", e)
        }
        audioRecord = null
    }

    private fun calculateRms(buffer: ByteArray, read: Int): Double {
        var sum = 0L
        for (i in 0 until read step 2) {
            if (i + 1 < read) {
                val sample = (buffer[i].toInt() and 0xFF) or (buffer[i + 1].toInt() shl 8)
                sum += sample.toLong() * sample.toLong()
            }
        }
        return kotlin.math.sqrt(sum.toDouble() / (read / 2))
    }

    private fun updateState(newState: State) {
        _state.value = newState
        listener?.onStateChanged(newState)
    }
}

    // ... 现有代码 ...
    // 悬浮窗隐藏时，停止唤醒监听（省电）
    wakeWordManager?.stopListening()
}
```

## 6. 关键问题与解决方案

### 6.1 APK 体积增加 ~50MB

**问题**：Vosk 中文模型约 50MB，打入 APK 后体积大增。

**方案**：
- 方案A（推荐）：直接打入 APK，50MB 对现代手机不是问题
- 方案B：首次启动从服务器下载，APK 保持轻量（但需要服务器）

### 6.2 模型加载耗时

**问题**：首次从 assets 复制到私有目录需要 1-3 秒。

**方案**：
- 在应用启动时异步加载，不阻塞 UI
- 加载完成后才开始监听

### 6.3 唤醒词检测不准

**问题**：Vosk 是通用 ASR，不是专门唤醒词引擎，可能漏检或误检。

**优化方案**：
1. **多候选词**：除了"小噜小噜"，也接受其他近似发音（如"小陆小陆"）
2. **置信度阈值**：检查识别置信度，只在置信度高时触发
3. **唤醒词 + 简短命令**：唤醒后直接进入命令识别，不用等待静音

```kotlin
// 在检测唤醒词时放宽匹配：
if (result.contains("小噜") || result.contains("小陆") || result.contains("xiaoluxiaolu")) {
    // 认为唤醒
}
```

### 6.4 电池消耗

**问题**：持续监听麦克风会增加耗电。

**优化方案**：
1. **VAD 前置过滤**：先用轻量算法判断是否有人声，没有人声时降低检测频率
2. **定时关闭**：如果一段时间没使用，自动关闭唤醒监听
3. **依赖 AccessibilityService**：只在用户主动打开悬浮窗时监听

### 6.5 误唤醒

**问题**：背景声音可能意外触发唤醒词。

**解决方案**：
- 唤醒词检测到后，悬浮窗展开显示"我在听"，用户确认后再收音
- 可以加一个确认步骤：唤醒后要求再说一次唤醒词

### 6.6 与现有语音输入的关系

**问题**：现有有"直接文本输入"和"输入法模拟"两种输入方式，语音唤醒后命令识别走哪条路？

**设计**：
- 唤醒词检测 + 命令识别统一走 **Vosk 本地识别**
- 识别出命令后，后续执行还是走现有的 `ChatViewModel.executeTaskLoop()`
- 百度语音识别可以**完全移除**（节省 APK 体积），或保留作为降级方案

## 9. 潜在风险与备选方案

| 风险 | 描述 | 备选方案 |
|------|------|---------|
| Vosk 中文识别精度不够 | 对某些命令识别不准 | 保留百度作为降级方案 |
| APK 体积增加 50MB | 用户可能抱怨 | 改为首次下载模型 |
| 后台监听被系统杀掉 | Android 系统省电策略 | 仅在使用悬浮窗时开启 |
| 误唤醒率高 | 背景噪音触发 | 加入确认步骤 |
| 模型加载失败 | assets 文件损坏 | 提供手动下载入口 |
